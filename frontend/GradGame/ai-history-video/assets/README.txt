Pharaoh character image for the history lesson video.

Default path used by the app:
  C:\Users\LENOVO\OneDrive\Desktop\ai-history-video\assets\pharaoh.png

To show the pharaoh character in the video (instead of only a gold screen):
  - Add an image file named:  pharaoh.png
  - Place it in this folder:  assets/

The image will be scaled and centered on the gold background. Use a PNG with a
transparent or solid background (e.g. a cartoon pharaoh, full body or upper body).
Recommended size: at least 400–600 px height so it looks clear in the video.

If pharaoh.png is missing, the video will still be generated with the gold
background and the narration voice only.
